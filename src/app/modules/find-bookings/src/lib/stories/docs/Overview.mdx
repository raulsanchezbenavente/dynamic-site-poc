# My Trips

<a href="https://figma.com/design/..." target="_blank" rel="noopener noreferrer">Figma Design</a> |
<a href="https://github.com/FlyrInc/dcx-ui-av/tree/main/projects/find-bookings" target="_blank" rel="noopener noreferrer">GitHub</a>

The **My Trips** module (`FindBookings`) displays a user's upcoming and past flight bookings. It manages authentication, data loading, pagination for upcoming trips, carousel navigation for past trips, and provides integration with modals for managing bookings (check-in, add trips, etc.).

## Basic Usage

The component is embedded in the main application via CMS configuration. It handles the following responsibilities:

- **Upcoming Trips**: Paginated list of future flights with check-in and booking management options
- **Past Trips**: Carousel view of completed flights
- **Loading States**: Skeleton loaders during data fetch
- **Empty States**: User-friendly messaging when no bookings exist
- **Error Handling**: Toast notifications for API failures

```html
<FindBookings #fbComponent />
```

```ts
export class FindBookingsComponent implements OnInit, AfterViewInit, OnDestroy {
  public config!: FindBookingsConfig;
  public isLoaded = signal<boolean>(false);
  public totalUpcomingTrips = signal<number>(0);
  public currentUpcomingPage = signal<number>(1);
  public upcomingTrips = signal<ManageBookingCardVM[]>([]);
  public pastTrips = signal<PastTripCardVM[]>([]);
  public isLoadingBookings = signal<boolean>(false);
}
```

## Key Features

### Pagination & Carousel

- **Upcoming Trips**: Server-side pagination (default 3 per page)
- **Past Trips**: Carousel navigation with auto-scroll detection

### Authentication Integration

Checks Keycloak authentication status:
- Authenticated → Load bookings via proxy service
- Unauthenticated → Show empty state with CTA to log in

### State Management

Uses Angular Signals for reactive state:
- `isLoaded`: Completion flag
- `isLoadingBookings`: API request in-flight
- `upcomingTrips` / `pastTrips`: Trip data
- `currentUpcomingPage` / `totalUpcomingPages`: Pagination state

### Toast Notifications

Provides user feedback for:
- Successful trip addition
- API errors (with localized messages)
- Booking actions

### Modal Integration

Triggers dialogs for:
- Adding new trips
- Managing bookings
- Confirming actions

## Translations
