# Icon list generator (IcoMoon → icons.json)

This folder contains the IcoMoon webfont assets and a small Node script used to generate a **friendly icon name list** for the CMS icon picker.

## What `extract-icons.mjs` does

- **Input:** `./icomoon/variables.scss` (generated by IcoMoon with `cssVarsFormat: scss_new`)
- **Logic:** finds SCSS variables that match `$icon-<name>:` and collects `<name>`
- **Output:** `./icons.json` (a JSON array of icon names)

The resulting `icons.json` is meant to be consumed by the Umbraco icon picker in:
`IbeCms\NS.Booking.CMS\App_Plugins\UI_PLUS\icon-picker\`
so editors can select icons by name instead of copying glyph codes.

## How to run

From the repo root:

```bash
node projects/design-system/assets/ui_plus/icons/webfonts/extract-icons.mjs
```

## When to run

Run this after updating the IcoMoon set (e.g., regenerating `icomoon/variables.scss`) to keep `icons.json` in sync with the available icons.
