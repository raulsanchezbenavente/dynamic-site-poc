# AV Design System
<span class="sb-tag">v 0.0.1</span>

AV Design System is the dedicated design system for the AV IBE product. It provides reusable Angular components designed to be used across all libraries and modules. These components are built with a strong foundation in usability and accessibility, while developing consistently with UX principles and brand guidelines, ensuring they meet the highest standards for modern web applications.

The system also integrates third-party libraries, such as Angular Bootstrap, to support advanced behaviors and interactions, leveraging tools like the Angular CDK for complex scenarios. This approach ensures flexibility and consistency across the entire product ecosystem.


## Accessibility Guidelines
Review the [A11y Component Guidelines](https://flyr.atlassian.net/wiki/spaces/IBE/pages/5453676691/A11y+Guidelines+on+components) for component-specific accessibility requirements that complement this design system.
