# Typography

Typography helps creating hierarchies, organize information, and guide our users through pages.

Main typeface: Red Hat Display (https://fonts.google.com/specimen/Red+Hat+Display)

[Design System Typography] includes some CSS custom properties (variables) in its compiled CSS for real-time customization without the need to recompile Sass. These provide easy access to commonly used values like theme colors, and primary font stacks when working in your browser’s inspector or general prototyping (ex.: storybook).
