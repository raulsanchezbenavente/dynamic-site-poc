# Minus Plus

<a href="https://github.com/FlyrInc/dcx-ui-av/tree/main/projects/design-system/src/lib/components/minus-plus" target="_blank" rel="noopener noreferrer">GitHub</a>

This guide focuses on how to use `minus-plus` in templates: how to configure it, which variants to pick, and how to handle accessibility labels for numeric steppers.

## Basic usage

The minus-plus component is configured through a `minusPlusVM` object. This object controls the form control, value range, and accessibility labels.

```html
<minus-plus [minusPlusVM]="minusPlusConfig" />
```

```ts
import { FormControl } from '@angular/forms';

minusPlusConfig: MinusPlusVM = {
  formValidationConfig: {
    formControlName: 'quantity',
    formControl: new FormControl(1),
    minValue: 0,
    maxValue: 10,
    isReadOnly: true,
  },
  labelledBy: 'quantity-label',
  statusLabel: 'Quantity',
  minusLabel: 'Decrease quantity',
  plusLabel: 'Increase quantity',
};
```

## Variants

### States

The component has several interactive states based on the current value relative to the min/max range:

<Canvas of={ComponentStories.STATES} />

- **Default**: Both buttons are enabled — the value is between the minimum and maximum.
- **At minimum**: The minus action is unavailable — the current value equals `minValue`.
- **At maximum**: The plus action is unavailable — the current value equals `maxValue`.
- **Single value**: Both actions are unavailable — `minValue` equals `maxValue`.

### Ranges

Different min/max configurations control the allowed range:

<Canvas of={ComponentStories.RANGES} />

- **Small range (0–3)**: For selections with few options (e.g., infants).
- **Standard range (1–9)**: Common range for passenger counts.
- **Large range (0–99)**: For quantity selectors with wide ranges.
