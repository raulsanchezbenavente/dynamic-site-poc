# TabsComponent

The `TabsComponent` is used to display and control a set of navigable tabs. It supports both projected content and dynamic configuration via an input.

## Basic Usage

```ts
tabsConfig = {
  items: [
    { id: 'tab1', tabTrigger: { label: 'Info' } },
    { id: 'tab2', tabTrigger: { label: 'Detalles' } }
  ]
};
```

```html
<tabs [config]="tabsConfig">
  <tab>Contenido de Tab 1</tab>
  <tab>Contenido de Tab 2</tab>
</tabs>
```

## Inputs

<tr>

## Best Practices
- Do not mix projected content and config-driven tabs.
- Use selectedId to sync external tab state.
- Ensure content is defined as string | null (not undefined).

## Supported Features
- Active/selected tab tracking.
- Disabled tab support via isDisabled.
- Emitting user selection via output.
